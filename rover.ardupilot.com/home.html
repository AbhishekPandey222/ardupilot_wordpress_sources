Rover Overview

Rover is an advanced open source autopilot for guiding ground vehicles and boats. It can run fully autonomous missions that are defined using mission planning software or pre-recorded by the driver during a manual run. Rover runs on the Pixhawk and a number of other supported autopilot boards. This is the platform that won the 2013 and 2014 Sparkfun Autonomous Vehicle Competition! That said, Rover is far more than just a toy - it is already being used in numerous applications (you can see a few examples including agricultural automation in our Video Demos).
