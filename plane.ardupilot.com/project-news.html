  • July 3, 2011: APM 2.20 released. Now supports the new standard magnetometer using the HMC5883L chip, which is autodetected. Also fixes problem with airspeed sensor autodetection in some setups. For now, to enable your airspeed sensor via the Mission Planner Configuration screen. Change the AIRSPD_ENABLE value to 1 to enable, then click "Write PIDs" to save. You can check that it's enabled correctly with the CLI Test "airspeed".
