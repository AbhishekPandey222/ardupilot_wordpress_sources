Mission Planner is free. You just need to download it and install it.  These sections explain how:
  • Windows Installer Use this for a simple fast way to install Mission Planner in your Windows XP, Vista, 7 or 8 system
  • Advanced Installation  Use this to install Mission Planner in a specific location or if you want multiple installations (requires WinZip) 
  • Apple computer Installation Install Mission Planner in an Apple computer with these instructions
Updates: If you are connected to the internet, Mission Planner checks for updates each day. You can accept the update or not. Updates are important because Mission Planner is always being modified  for enhancements, to fix bugs and to support the various auto pilots, Planes, Copters, and Rovers.  Updates will include new defaults for parameters for the various vehicles.  If you update the firmware in your auto pilot you would be wise to update Mission Planner to the latest version. During the update you will see various screens and scrolling data.  These screens are not important to the normal user - just wait for Mission Planner to re-start.
