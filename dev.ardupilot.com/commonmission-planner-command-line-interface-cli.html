Using the Mission Planner serial terminal

The Terminal tab is visible on the top icon row when “Advanced View” is checked in the Config tab. The Mission Planner serial terminal is the best a way to see text data coming from APM via its USB port and to type commands into the CLI. You can can also run tests and download and analyze log files from this screen.
