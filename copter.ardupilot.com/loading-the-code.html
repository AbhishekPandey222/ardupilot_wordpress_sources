The easiest way to load the latest code is by using the Mission Planner much like you would for a quad . You can download the latest Mission Planner here . The only difference being that after you click on the "Firmware" button, you click on the helicopter icon. Alternatively you may download the source code from google code using a tool such as Tortoise git, load the code into Arduino IDE and press the Upload button as described in these slightly out-of-date instructions but this is for advanced users who want to customise the code in some way.
