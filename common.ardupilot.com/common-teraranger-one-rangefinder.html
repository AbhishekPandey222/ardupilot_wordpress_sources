The TeraRanger One rangefinder is a lightweight, high-performance distance measurement sensor based on infrared Time-of-Flight (ToF) technology. It is much faster than ultrasound and far smaller and lighter than laser-based systems. Its small size, high performance and low power consumption make it ideal for modern robotic applications, drone operations and automation. More technical information about ToF sensors can be found here. [note]This rangefinder is only supported on the Pixhawk via I2C protocol. Only the TeraRanger One sensor is compatible with this protocol.[/note]

Connecting to the TeraRanger via I2C

The TeraRanger One sensor can be easily connected to the Pixhawk with the TeraRanger I2C adaptor. Then it’s nearly a plug&play solution:
  1. Solder the alimentation for the TeraRanger I2C adaptor: [caption id="attachment_5829" align="alignnone" width="399"] TR-I2CAD Power Cables[/caption]
  2. Connect the I2C Adaptor to the Pixhawk via a DF13 4S cable include into the Pixhawk box, and the TR-One to the I2C Adaptor: [caption id="attachment_5699" align="alignnone" width="1654"] TR-One connection to Pixhawk with TR-I2CAD[/caption]

Setup in Mission Planner

To use the TeraRanger One sensor as rangefinder, connect with Mission Planner and open the Config/Tuning menu. Go into Full Parameter List section and set:
  • RNGFND_MAX_CM: 1400 (14m). This parameter sets the maximum distance that the TeraRanger One is used by the Pixhawk.
  • RNGFND_MIN_CM: 20cm. This parameter set the minimum distance that the TeraRanger One is used by the Pixhawk.
  • RNGFND_TYPE: 4

Testing the sensor

The distances read by the TeraRanger One sensor can be displayed on the Mission Planner Flight Data menu. You have to double click on the left bottom windows where quick values are displayed. Then choose “sonarrange”. [caption id="attachment_5693" align="alignnone" width="590"] Mission Planner: Setting to display rangefinder data on the Flight Data screen[/caption]
