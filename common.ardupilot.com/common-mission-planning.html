This section contains articles about Mission Planning; creating automated missions that will run when the ArduPilot is set to AUTO mode. [pagelist_navmenu] [note]The examples demonstrate mission planning using Mission Planner, a fully-functioned Windows-based Ground Control Station (GCS) that you can download and install from here. [caption id="attachment_4959" align="alignnone" width="474"] Mission Planner: Flight Plan Screen[/caption] The main concepts will also apply to other Ground Control Stations (although the details of the user interface will differ).
