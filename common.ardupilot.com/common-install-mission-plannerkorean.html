미션 플래너 설치하기

미션 플래너는 윈도우 운영체제를 위한 무료, 오픈소스 소프트웨어입니다. 다음 설명서는 여러분의 지상통제 컴퓨터에 미션 플래너를 설치하는 방법을 안내할 것입니다.

최신의 미션 플래너 설치파일 다운로드 하기

미션 플래너 다운로드 페이지 로 가서 "Mission Planner - MSI - Permanent link to latest" 를 클릭하여 다음 페이지에서 다운로드 버튼을 누릅니다. 또는 다음 링크에서 MSI를 직접 다운로드할 수 있습니다.

RUN THE INSTALLATION UTILITY

마이크로소프트 설치파일 (.msi)을 실행하여 설치 도구를 엽니다. 지시에 따라 설치과정을 마칩니다. 설치도구는 자동으로 필요한 소프트웨어 드라이버를 설치합니다. DirectX 설치 오류가 발생하는 경우 DirectX 다운로드 센터에서 DirectX 플러그인을 업데이트하십시오. 아래 보이는 그림과 같은 경고창이 뜨면 "이 드라이버 소프트웨어를 설치"를 선택하고 진행하시기 바랍니다..   미션 플래너는 일반적으로 C:\Program Files (x86)\APM Planner 폴더 혹은 C:\Program Files\APM Planner 폴더에 설치됩니다. 여기가 로그파일이 위치한 곳입니다. 설치과정에 선택한 옵션에 따라 미션플래너 아이콘이 생성됩니다.

미션 플래너 열기

설치를 마치면 아이콘을 더블클릭하여 미션 플래너를 실행합니다. 미션플래너는 자동으로 최신의 사용가능한 업데이트를 알려줍니다. 항상 최신 버전의 미션플래너를 사용하십시오.
